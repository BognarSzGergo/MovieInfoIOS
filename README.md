# MovieInfoIOS
 
The main page displays 4 list:
 - Now Playing Movies
 - Upcoming Movies
 - Top Rated Movies
 - Popular Movies

When you tap on an item, the Details page opens.

The Details page displays:
 - Title
 - Backdrop
 - Genre
 - Release Year
 - Run Time
 - Overwiew
 - Rating
 - Trailers (links to youtube videos)
 
 The data is downloaded using The Movie Database API.
